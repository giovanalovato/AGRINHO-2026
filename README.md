# 🌱 Agrinho 2026 - Agro forte, futuro sustentável

Este projeto foi desenvolvido para o Concurso Agrinho 2026, com foco em sustentabilidade no campo e uso consciente da tecnologia na agricultura.

## 💡 Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## 🌍 Objetivo
Demonstrar como a tecnologia pode ajudar a agricultura a ser mais sustentável, reduzindo o uso de água e preservando recursos naturais.

## ⚙️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 💧 Funcionalidade
O projeto contém um simulador que mostra a economia de água possível com o uso de tecnologias agrícolas.

## 🚀 Como executar
Abra o arquivo `index.html` em qualquer navegador.
